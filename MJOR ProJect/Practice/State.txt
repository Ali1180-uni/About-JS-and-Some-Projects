Types:
1. Stateful Protocol -> Save Status & Session info in Server -> FTP (File Transfer Protocol)
2. Stateless Protocol (By Default) -> Never Save Status and Session info in Server -> HTTP (Hyper-Text Transfer Protocol)

-> Express Sessions make our Sessions Stateful
